STYLE

Style is an attribute used to add styles to an element, such as color, font, size, and more.

Example:

This text is red wow!! :)

CODE:

< p style="color:red">This text is red wow!! :)

Example 2:

This text is brown wow!!

Code: < p style="color:brown;font-size:200%">This text is brown wow!!


Background coloring:

Background color is red with width 40%.

Code: < p style="background-color:red;width:40%"";>Background color is red with width 40%.
